# YDeck Company Profile Layouts

This template contains 25 layout options. It is a layout design system, not a fixed 25-slide deck. Use only the layouts that fit the user's deck type, selected flow, source material, and requested slide count.

## Chart guidance

For metric, comparison, map, timeline, forecast, risk, finance, and dashboard layouts, request a chart from create_chart when structured data exists. Use the returned ECharts static SVG as an inline visual block. Do not use browser-side chart scripts in slide HTML.
