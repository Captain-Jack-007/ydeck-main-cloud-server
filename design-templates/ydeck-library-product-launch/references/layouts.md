# YDeck Product Launch Layouts

This ydeck-library template contains 15 professional slide recipes. Use these layout ids as the allowed vocabulary for this template.

## `yl_title_statement`

Slide 1 role: Opening thesis.

Use: Set the strategic point of view with one confident statement.

Required slots:

- eyebrow
- title
- primary content
- supporting detail
- footer with slide number

## Chart guidance

For metric, comparison, timeline, risk, operating model, and financial slides, request a chart from create_chart when structured data exists. Use the returned ECharts static SVG as an inline visual block. Do not use browser-side chart scripts in slide HTML.
