# YDeck Book / Chapter Lesson Deck Layouts

This template contains 25 layout options. It is a layout design system, not a fixed 25-slide deck. The agent must choose the recommended flow that fits the user's request, then use only the layouts needed for the requested slide count.

## Design Rule

Every layout is a classroom artifact. Do not generate a repeated deck shell with different titles. Preserve the artifact listed for each layout.

## Layout Options

## `yl_book_chapter_lesson_chapter_title`

Option 1 role: Chapter title.

Artifact: Open-book chapter cover.

Use: Use this layout for chapter title content in a book chapter lesson deck.

Required slots:

- left page title
- right page chapter mark/source note
- book spine
- footer

Rules:

- Preserve the open-book chapter cover composition.
- Use the template palette semantically.
- Include a visible student, teacher, or reader action when the slide is instructional.
- Do not place charts or KPI cards on the cover.

## Diagram guidance

Use diagrams only when they teach structure: map, timeline, evidence chain, claim ladder, theme layers, sentence surgery, seminar flow, or answer states. Do not use browser-side chart scripts in slide HTML. Do not add charts or bars for decoration.
